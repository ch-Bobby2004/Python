Pandas is a popular Python library used for data analysis and manipulation.

Think of it like a powerful tool for working with tables (similar to Excel or SQL), but inside Python.

It helps you load, clean, analyze, and manipulate data
Works especially well with structured data (rows and columns)

Main data structures:
Series → a single column (like a list)
DataFrame → a full table (rows + columns)


What you can do with Pandas:
Read data from files (CSV, Excel, JSON)
Filter and sort data
Handle missing values
Perform calculations and statistics
Group and summarize data
In simple terms:

Pandas = Excel-like data handling + Python power
